The most recent (last) record for each active FLEX option.

AttributeValue

FieldTypeComment

BODY​

FieldTypeComment
eventIdintunique id of the event assigned by SpiderRock, subsequent messages on the event will use the same id
eventTypeenum : FlexEventTypetype of event this message represents, e.g., RFQ, QTE, LST
netTimestamplongtimestamp of when this message was received
packagePricedoubletotal price of the flex trade inclusive of all legs
prtExchenum : OptExchexchange code that produced this flex event
prtRootstring(6)security symbol of the FLEX option
requestIdintrequest identifier associated with this trade assigned by the exchange (these appear to not be 100% reliable, particularly for PHLX)
srcTimestamplongtimestamp from the exchange of when this message was sent
totalQuantityinttotal quantity of the flex trade
