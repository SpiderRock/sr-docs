Release Notes

September 27th, 2025 Release Notes​

Message and enum differences can be found here.

SpiderRock Connect:​

  • Added SMH, SOXL, and SOXX to late-close symbol list.
  • Addressed calculation issue with stale quote probability records.
  • Addressed issue with closed orders appearing in the 'Active' tab of ticket blotter.
  • Addressed issue with liveImpliedQuote records for NDX and RUT outside of regular trading hours.
  • Allow primary exchange control for stockExchangeImbalanceV2.
  • Correctly populates values for fitAvgAbsErr and fitAvgErr in liveSurfaceCurve.
  • Fixed incorrect print volume values in stockCloseMark.
  • Modified default autohedge target for options-on-futures hedging.

Citrix Tools:​

  • Order Manager - Addressed issue where stock limit trigger price is ignored when releasing orders.
  • Order Manager V2 - Properly greys out menu items and allows cancels for view-only users
  • Risk Manager - Added weighted $Ve.
  • Risk Manager: Basket Blotter - Allow size adjust to persist in presets.
  • Symbol Viewer: Ticket Blotter - Addressed issue with closed orders appearing in the 'Active' tab.

Trade​

  • Added a criteria pulldown to the FLEX order ticket.
  • Added a new menu bar that positions and risk data for a single ticker.
  • Added ability to create a custom expiry to the FLEX option montage.
  • Added FLEX regular and FLEX delta-adjusted close (DAC) identifier to order cards and auction cards.
  • Added positions column to option montage.
  • Can now convert listed options to FLEX options in the order ticket.
  • Can now toggle between FLEX criteria on the FLEX option montage.
  • Optimized spacing on options montage and readability on montage footer.
  • Supports FLEX DAC order types.

Portal​

  • Bump context query getMsgs limit to 50,000.
  • Removed histograms.
