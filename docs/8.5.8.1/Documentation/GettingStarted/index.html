Getting Started

Introducing SpiderRock Connect​

Welcome to SpiderRock Connect (aka version 8 or V8), the latest version and next generation SpiderRock trading platform.  SpiderRock Connect supports the legacy Citrix tools and an upgraded version of SRSE, but also includes re-imagined web-app tools, our new MLink API and the SpiderRock ATS enabling RFQ-style trading of listed and FLEX options. Please contact our Sales department at learn@spiderrock.net to discuss how to get started with SpiderRock Connect.

    If you are an existing client that has already been onboarded onto SpiderRock Connect and you have questions about obtaining credentials to a specific API, please reach out to our Client Support Desk at cs@spiderrock.net.

Accessing GUI Tools​

Legacy Tools / Citrix Workspace​

The SpiderRock Client Support Desk will generate a password for the user, which will be sent via secure PDF.

Users can access legacy tools through Citrix by logging in at https://sms.spiderrock.net/vpn/index.html.

Next-Generation Tools / WebApp Tool Suite​

SpiderRock Connect can be accessed either via a web browser at https://spiderrockconnect.com/home OR by our SpiderRock Connect desktop application which can be downloaded here:


  1. Users should contact the SpiderRock Client Support Desk to create a unique SpiderRock ID and generate an invitation that will be sent to that user’s email address.
  2. Upon receiving the email, the user will follow the link provided to complete their registration.
  3. Once redirected to your browser of choice, a login prompt will appear. The username field should already be filled with the username associated with your user in SpiderRock’s system.
  4. Insert the password that would have been established with SpiderRock during the onboarding process.
  5. Choose your verification method of choice. Verifying with an authentication app is recommended and these instructions will follow that workflow. If you choose to verify with a phone text, follow the prompts given.
  6. Enter your authentication code and click continue.
  7. After completing these steps, the user will be placed on the new Launcher browser page where they can launch any tools they have been configured to access.

For additional assistance on your log-in information or difficulties with the Documentation site’s download, contact the SpiderRock Client Support Desk. If assistance is needed with the Tool Suite’s installer and depending on the nature of the issue, contact your IT department or the SpiderRock Client Support Desk at cs@spiderrock.net.

Accessing SpiderRock via APIs:​

Users can access the SpiderRock system in several programmatic ways using APIs.

  • Access via MLink via REST

  • Access via MLink via WebSocket

  • Access via SRSE

  • Access via FIX

Testing and Using the Sandbox​

SpiderRock Connect provides access to a test environment called the Sandbox. In this environment, there are pre-loaded datasets to help users test connection and applications in a predictable, safe environment with no risk of reaching the outside markets. To connect to the Sandbox, follow the instructions on our V8 Sandboxes here.
