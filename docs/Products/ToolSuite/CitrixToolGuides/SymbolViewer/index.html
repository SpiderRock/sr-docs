Symbol Viewer

The SpiderRock Symbol Viewer is a live order entry and risk management tool at the symbol level, allowing users to trade simple DMA style as well as complex equities, options and futures orders. The tool provides access to all SpiderRock algorithmic execution logic and displays live prices and analytics for all products supported by the platform. It also displays real-time position risk records and has graphical capabilities to display option analytics as well as position greeks. This document is a concept guide and provides an overview of the tools main features. For more details, please contact the support desk at cs@spiderrock.net or reference other related concept guides.

The table below is a description summary of each tab available via the Symbol Viewer. Only the actionable tabs (RiskView and Spreads) and chart tabs are described in this document. For a complete store list of available metrics, please contact the support desk at cs@spiderrock.net.

RiskView Tab​

The RiskView tab is an option montage screen, which provides pricing information (NBBO prices, implied vols, alpha probabilities), dividend streams, earnings, interest rates, hard to borrow rates, greeks and many other metrics users can select from right-clicking on any column header below the tab selectors.

The window has three distinct parts: the main symbol panel, the summary information panel, and the detail information panel as shown below.

The main symbol panel (upper right corner) allows users to select one or several SpiderRock accounts, an asset/symbol, and “what if” overrides to drive the analytics in the detail window of the tool. The Refresh button loads the data.

Server Panel​

Users can customize all above values. These values are an example of possible choices.

  • The detail information panel is highly customizable by the user. Screen colors and metrics are all configurable by right-clicking on any column header right below the tab selectors.
  • Users can launch a simple order entry ticket (as shown on the next page) for stocks, futures and options by double-clicking on any bid/ask prices.
  • Users can also launch complex orders in stocks, futures and options by right-clicking on any bid/ask price, or highlighting several strikes and right-clicking. This action will display the following menu:

  • User can choose to create a new option or multi-leg ticket or add to an existing option or multi-leg ticket
  • Radio buttons (1, 2, 3, 4, 5, 6) are available in this menu to recall pre-saved settings for option and multileg tickets
  • User can also select a closing strategy, launch a ticket on NBBO quotes, or launch a ticket in a corresponding underlyer

Users can view positions in a particular symbol from that screen and corresponding greeks. Users can also simulate positions in the test columns and visualized the corresponding simulated greeks as they turn pink from the initial orange color.

Launchign Order Entry Tickets:​

Quick Tickets: A single order can be launched from left-double clicking on any bid/ask prices (or implied volatilities) from the options montage screen, or any bid/ask prices for the underlying instrument in the upper right corner of the screen. A number of settings are available to users to launch an order, including order size, price or volatility limit, order type, limit class, max visible, number of exchanges (when posting liquidity), cancel overlap, auto hedge strategy, time-in-force, and more.

Ticket Blotter: Order baskets can be launched from highlighting any strikes on the options montage screen across multiple expiries and multiple symbols if desired, then right-clicking and choosing Spider Option Ticket (New). This will populate the ticket blotter with all selected series. All settings from the quick ticket and more are available to users to launch an order basket. In addition, users can apply risk groups that act as guardrails when executing the basket. These risk groups can be expressed as vega limits, premium limits, contract limits and/or dollar delta limits.

For further information, please see the Order Entry Tickets Concept Guide or contact the support desk at cs@spiderrock.net.

Spreads Tab​

  • The spreads tab allows users to apply filters to spread feeds so that corresponding spreads waterfall in the display window. Users can then select a particular spread from the display window and launch a spread order entry ticket to trade the spread.
  • SpiderRock currently consumes the CBOE, ISE and PHLX spread feeds for equities, and the CME, ICE and CFE spread feeds for futures.
  • Users can select a combination of different filters. The main filter buckets are surface, edge, quantity, probability, spread type, spread ratio, moneyness, time, nature of the spread components, source, and exchange.
  • Once filters are selected, and corresponding spreads waterfall in the display window, users can double left-click on a line (see red arrows below), which populates an order entry ticket.
  • The order entry ticket allows users to select order type, order size, order price, limit class, probability limit, auto-hedge, time-in-force, leg exposure and more. For more information on the spread ticket, please contact our customer support desk at cs@spiderrock.net.

VolSurf Chart​

  • The VolSurf Chart tab allows users to visualize the SpiderRock calculated volatility surface, which is fit dynamically several times per second through the bid/ask channel for a particular symbol/expiry.
  • Puts and calls are visually represented by the colored cones formed by the vertical lines. The height represents the width of the market. If the Current view setting is set to Best (best market), the user will typically see the smooth surface fall between the prevailing bid/ask spread of each strike.
  • In the Charting option panel, users can select the expiration from the pull-down menu or click an expiration from the Expiration panel. Users can also select current market views (All, Best, etc), overlay the previous vol surface, overlay user-defined theo vols to the SpiderRock surface, and also display a diagram of the current positions and daily volume in the particular symbol/expiry under the surface chart.
  • In addition, users can select different X-Axis and Y-Axis display options: view all strikes or the ones that are at +/- 1, 2 or 5 STD from the ATM strike. View outright vol, vol skew (centered on 0), vol difference, or price difference (vol difference * vega).
  • For additional metrics related to the calculation of the surface, users can click on the 'More' button (see below)

VolTerm Chart Tab​

  • The VOLTERM chart tab allows users to see a visual representation of the actual term structure (uncensored and/or censored for earnings) as well as a fixed-term structure of the ATM volatility for a specific symbol across a specified length of time.
  • The X-Axis displays the length of time in days and years. The time display choices are 3 months, 6 months, 1 year, or 2 years.
  • The Y-Axis represents ATM volatility in volatility points.
  • Users can add a custom event to the chart and see how such events would impact the term structure by choosing the magnitude of the move and the time frame, in years.
  • User can also overlay a “big picture” curve by adjusting the short-term vol (ST Vol), long-term vol (LT Vol), decay (Decay) and earning move (eMove). This “big picture” or theoretical curve can be censored or uncensored.

ATMVol Chart Tab​

  • The ATMVOL chart tab allows users to have a 5-minute snapshot visual representation of the SpiderRock ATM volatility surface for up to 10 trading days.
  • The baseline (shown in black) is a line drawn showing the majority of time the ATM Vol spent at any one value. The vertical “T bar” line shows the highest and lowest values during that time period.
  • The X-Axis represents the time of day (in EST). The Y-Axis represents the values denoted below.

| --- | --- | | ivXX | Implied volatility of fixed strike, based on prior day settlement | | uPrc | Underlying price | | sDiv | Continuous dividend rate | | Slope | Volatility surface slope | | uOff | Underlying offset (for Futures) | | vsFV | This is the variance sweep for forward volatility | | log(iVol/SPY) | Implied volatility of the selected symbol over SPY implied volatility in log terms | | iVol - SPY | Implied volatility of the selected symbol minus SPY implied volatility | | log(ivXX/SPY) | Implied Volatility of fixed strike over SPY implied volatility in log terms | | ivXX - spy | Implied Volatility of fixed strike minus SPY implied volatility | | log(uPrc/SPY) | Underlying price of selected symbol over SPY price in log terms |

  • The underlying price (last sale) and slope are captured during the same period of time. The slope is calculated as a rate of change between the volatility levels at the +/- ½ moneyness strikes.
  • We also allow the ability to override the default scaling by hitting the 'Override Ranges' button

Risk Chart Tab​

  • The Risk Chart is designed to show a user’s current position Greeks for a selected symbol across all expirations (default) or a specific expiration in that symbol.
  • It also allows users to shock Greeks PnL across underlying symbol price moves.
  • The X-Axis line represents the underlying price, with the ATM price being in the middle of the grid (0.0%). Max range specifies the range of values on the X-Axis.
  • The Y-Axis line represents the values denoted below:
    • $pnl - Black line
    • Delta – Blue line
    • Gamma – Green line
    • Theta – Brown line
    • Vega – Purple line
  • Other options are available across the top of the chart, such offsets in vol (pct), rate (pts), sdiv (pts) and days, expiration selector, include underlier or not, and ATM delta intervals.
  • The composition box displays the user’s position in option contracts, futures contracts and shares.
