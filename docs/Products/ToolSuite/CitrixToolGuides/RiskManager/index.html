Risk Manager

The SpiderRock Risk Manager is a live portfolio risk management tool allowing users to analyze and quickly shape portfolio risk. Users can create custom risk slides to understand the risk sensitivity of a portfolio and address areas of concern with risk-reduction capabilities.

The Risk Manager also includes a live market-scanning application designed to identify potential risk-opening opportunities based on rich user-defined filter settings.

General Tool Layout​

This tool is composed of three different tabs:

TabDescription
The Risk Aggregations TabAggregates risk across a variety of measures for all asset classes composing the portfolio. The tool has 2 general aggregation lists and additional filter buckets across the top. The tool is also organized as a split window where the detail of each aggregation line can be exposed in the bottom part of the grid.
The Span Aggregations TabProvides a Standard Portfolio Analysis of Risk (SPAN) methodology evaluate risk under a broad range of ‘what-if’m arket scenarios. The customary 16 slides based on an underlier price and vol change can be displayed following the CME exchange guidelines or can be modified with user input, based on price/vol settings.
The Risk Detail TabProvides all risk metrics available at the series level for a selected account(s) in a non-aggregated capacity. Clients can easily export position details at the series level into Excel.

Risk Aggregations Tab​

The Risk Aggregations tab allows a visualization of current portfolio risk (positions plus day trades) in multiple aggregation buckets. Two aggregation lists are available across the top which enables any portfolio manager to analyze and monitor risk across accounts, symbol, industry, expiration etc. This tool includes all sources of trades (GUI Tools, FIX, SRSE API).

The tab launches on a grid (Aggregated Risk Pane) which is a default view of the client's risk across the portfolio.

In addition, clients can click on any aggregation line from the Aggregated Risk Pane to display the individual trades composing this agg line in the split window below (Constituent Risk Pane).

Creating Risk View​

In addition to the default view, clients can easily create their own custom views by clicking on the + sign to the right of the drop-down list which prompts an Add new Aggregation View window. Once the new view is named e.g. PnL View as displayed above, clients can customize their view. Refer to the Column Properties Features section for column selection details.

Selecting Aggregation Buckets​

The Agg selectors (see red arrows), allow clients to aggregate fills using two unique values. If two aggregation values are selected, fills are first aggregated using the value from the first dropdown list and then using the value from the second dropdown list to the right. If the second value is set to 'None', then fills are aggregated using the first value.

The Totals across the top also follow the same aggregation selection.

In addition, clients have access to a number of filters across the top where they can further refine their view of risk. For example, a client may be interested to look at a specific symbol or a specific expiration.

All columns can be sorted by double-clicking the header (double left-click ascending, double left-click again, descending).

Additional Filters​

Additional filters are available right above the tabs enabling clients to further examine specific areas of their portfolio. Clicking on any filter will bring up a list of values available for a particular filter. Any values can be checked using the square check box to the left. To remove any previously checked filters, clients can click on the funnel in the bottom right of the tool.

Furthermore, often used filters can be saved using one of the eight available radio buttons to the left of the blotter box as displayed below.

    Note that this is in addition to any created custom views.

Totals​

Configurable totals boxes are available and displayed above the grid and below the tabs.

  • Right-clicking on any of the total boxes, will open a window of available values that can be checked and added as a total box.
  • Totals are responsive to any filters used across the top.

Column Properties Features​

Columns can be moved left or right by clicking and dragging columns within the Aggregated Risk Pane. To adjust the order of all columns at once, open Column Properties by right-clicking on any headers or by clicking on the settings box (top right corner) and then click on Column Properties. In the Column Properties window click the ‘Reorder’ box which will display all columns in the left panel. Clients can drag and drop all available columns into a specified sequence.

To change the width of individual columns, left-click the divider between column headers and drag left or right. The width of all columns may be adjusted from the Column Properties menu.

From this window, the user can select columns and filters to display.

  • The ‘Caption’ header shows the name of the column.
  • Users can check any values to populate the grid.
  • Right-clicking anywhere in this window allows clients to check all or uncheck all values.
  • The ‘Font’ column allows users to set the foreground and background colors for each column.
  • The ‘Description’ column provides specific calculations and further information.
  • The ‘Search’ box allows the user to find a metric without scrolling through the entire list. The search can be done at the ‘Group’, ‘Caption’ and ‘Description’ level by checking the appropriate box to the right of the search window.
  • The (+) and (-) buttons allow the user to adjust the width of the columns in the Risk Manager. The Auto Width button will resize columns to the smallest width required to display all data.

Creating Risk Scenarios​

Clients can compose up to eight distinct risk scenarios by clicking on the Scenarios button displayed below (see red box). These risk scenarios enable clients to shock their portfolios either in price terms, volatility terms, change in expiration dates, change in rates, or percent change in dividend.

Once created, a scenario can apply to any Greeks, as well as PnL, and can be selected using the Column Properties window as described on Page 8.

Creating Market Scans​

Market scans are created from the Macros button as displayed below. Macros were designed for clients to scan the market in real-time based on a comprehensive set of filters to define the stock universe, the option universe, as well as edge criteria if desired. These scans can be used as a trading ideation tool and can also be used to increase or decrease risk across the portfolio.

Please refer to the Macro Scanner Concept Guide for more information.

Risk Shaping/Basket Trading Functionality​

Opening/increasing risk can be done in two ways:

  • Market scans (macros) can be used to put on new positions.
  • Basket Trading

Clients can launch a Basket Blotter for single and multiple leg orders by clicking on the Blotter or Blotter (MLeg) buttons as displayed below:

A blank blotter can be populated from copying orders from Excel and clicking on the Import button of the Blotter (red rectangle below).

The Excel example below is showing the mandatory fields required to populate a Blotter. There are a number of non-mandatory fields that can be added to create orders. For a list of these fields, please contacts us at cs@spiderrock.net.

The orders imported from Excel are shown below in the Risk Manager Basket Blotter. Once imported, clients can set algo parameters across the top of the Blotter. See our Order Entry Concept Guide for more information.

Closing/Decreasing Risk​

Custom market scans can be used to close risk against some/all positions in the portfolio by setting the position type of the macro to 'CloseExact' or 'CloseSmallRR'.

Users can also close highlighted positions in the grid by right-clicking and selecting 'Close Position'. This will open a window where the user can select 100%, 75%, 50%, 25% (see example below). Users can choose whether to include stock, options, and futures by checking the appropriate box(es). Once selected, clicking OK will launch a blotter with the corresponding orders to close these positions. If the user wants to override the size once the Risk Blotter is open, they can do so by adjusting the "Size/Adjust" Factor amount to any percentage.

Span Aggregations Tab​

The Span Aggregations tab provides a Standard Portfolio Analysis of Risk (SPAN) methodology to evaluate risk under a broad range of ‘what-if’ market scenarios. The customary 16 slides based on an underlier price and vol change are displayed.

The tool captures all live order activity in real-time. All columns can be sorted by double left-clicking the column headings (double left click ascending, double left click again, descending).

Users can isolate one account or select multiple accounts to run span aggregation risk. See red box on account dropdown menu.

The Agg selectors (see red arrows) allow the user to aggregate risk by two unique criteria from each of the Agg dropdown menus. If two aggregation criteria are selected, the Risk Manager will first aggregate by the left Agg dropdown menu and then by the right Agg dropdown menu. By selecting 'All' from the right Agg dropdown menu, only a single criterion will be considered for aggregation.

Users can select any available Span Metrics (Price, Delta, Gamma, Theta, Vega and WtVega) to view span aggregation risk across selected accounts.

The Span Aggregation tab is split between the Aggregated Risk Pane and the Constituent Risk Pane (see below).

Source Settings​

Users have the choice between a default Exchange mode and a Custom mode. When Exchange is selected, all price and vol settings will default to set values (CME Group). When Custom is selected, the following price and vol settings are available (see below).

Span Metrics​

The span calculation can apply to any of the following metrics: Price, Delta, Gamma, Theta, Vega and WtVega.

Worst possible loss values are clearly identified within each scenario in the red cells above.

The maxLoss column reflects the worst possible scenario from all 16 shocks.

Additional filters are available at the top of Risk Manager to target risk with more granularity.

Risk Detail Tab​

The Risk Detail tab contains a single row for every strike and underlying position in the selected account in a non-filtered capacity. Users can easily export the content to Excel for further analysis. To highlight the entire grid, click on the box highlighted in green below. Then hit Ctrl + C to copy and Ctrl + V to paste into Excel.

To select an individual column, left-click the header. To select a contiguous group of columns, left-click the header of the first desired column, hold Shift, and click the last desired column. To select a non-contiguous group, click the header of the first desired column, hold Ctrl, and select each desired column in turn. To select all columns, left-click the blank column header at the far left.

Once selected, columns may be copied and pasted into programs like ‘Excel’ using Ctrl-C and Ctrl-V respectively. Selected columns will be transferred along with their column headers.
