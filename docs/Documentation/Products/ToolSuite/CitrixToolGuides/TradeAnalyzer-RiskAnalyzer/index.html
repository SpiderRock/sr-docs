Trade and Risk Analyzer

Both the Trade Analyzer and Risk Analyzer have similar functionalities and are intended to provide the client with their own trade and risk records for a maximum period of 5 years of history. The tools are designed as an Excel pivot table and allow clients to create customized reports.

Pivot Table​

Upon launching the Risk Analyzer, the following settings are available:

The Risk History Pivot Grid is the tab used to custom build any reports using the pivot functionality of the tool.

The Risk History Detail tab displays all available records for the account and Date Range selected.

The client can aggregate data across several accounts using the dropdown menu to select multiple accounts.

Tickers: Clients can select a symbol(s) using the tickers box. Each symbol should be delimited using a comma.

Date Range: Clients can select a start date/end date or use the date range box as displayed below which gives 4 different options: Today, DateRange, ThisWeek, ThisMonth.

Hide Field List: If checked, the field list on the right will disappear providing additional room to view custom reports.

Default View: Clients can create as many views/reports as desired by clicking on the plus button. This will prompt an Add new Pivot Grid window to appear on the screen. Clients can then name the report and save it. This newly created report will appear in the drop-down list. Clients can also delete reports using the minus sign.

Accnt: Clients can either type their account(s) in the box or use the dropdown menu to select their desired account(s).

Refresh button: To run and update on an existing report, clients can click on the refresh button to get the latest data available (see red arrow below).

Settings box: The settings box allows clients to load or save settings as well as customize their columns.

Risk Analyzer Pivot Table​

The Pivot Table section is composed of three different areas:

  • Fields List (see red box below) - includes all the dimensions and measures that are part of the SpiderRock Risk Cube Table (available via SRSE Risk). When users click on the plus sign, all metrics associated to this particular dimension/measure appears with a small clickable box to the left (see red arrow below). Clients can select any metric by clicking on the check box or by dragging the metric to the appropriate section below (highlighted in green).
  • Report Builder (see green box below) - This section is composed of the Report Filters, Column Labels, Row Labels, and Values. These sections allow clients to create custom reports by filtering and summarizing data.
  • Pivot Grid (see blue box below) - This is the display window for the newly created report. By right-clicking on the gray box (see green arrow), a client can edit the formatting of columns and headers as well as export to Excel.

Risk Analyzer Fields List​

Clients have access to a description of all dimensions and measures in the Risk History Detail tab by right-clicking on any column headers. This will bring up the Column Properties window for all available definitions. Clients can also use the search field to look up values by Group, Caption or Description.

Creating a Report​

In the example below, the client selected an account and a date range.

The client clicked on the plus sign on the Dimensions - Security field to display the various values.

  • SecType was selected as a report filter.
  • To filter on SecType, the client double-clicked on the SecType field under the Report Filter section and clicked 'Select Items'.
  • A Filter Items window appears as displayed below which enabled the client to select Call and Put.

Under the same Dimensions - Security, both Symbol and Expiration were selected as raw labels.

The client clicked on the plus sign to the left of Measures - Live Risk to display all values.

  • The client selected posDelta, posVega and posTheta as the values to be summed.
  • Double-clicking on any of the selected values would allow the client to change the calculation from a sum to a count, average, others.

The corresponding report appears under the Pivot Grid to the left and can be exported to Excel.
