Ticker Ribbon

The Ticker Ribbon is an interactive component that allows the user to change tickers, launch an equity order, displays relevant market data, and provides access to the client account and profile.

Ticker Search​

Users can click in the ticker search box and either start typing a ticker or a company name to find their desired security.

Bid and Ask Buttons​

The bid and ask buttons display the underlying security’s bid and ask prices. Clicking on the bid will launch a sell equity order ticket, and clicking on the ask will launch a buy equity order ticket.

Lightning Charts​

The two lightning charts display intraday price movements of the underlying security today and the previous day. The left lightning chart displays the previous trading day’s intraday price movements and the right lightning chart displays the current day’s intraday price movements.

Earnings and Dividends Data​

This is where applicable, earnings, and dividends data will be displayed for the underlying security. In addition, the user can click on the “Earnings” or “Dividends” buttons to open a more detailed visual.

Account Selector Menu​

Clicking on the account box will open an account selector menu. Typing inside of the search box will allow the user to find accounts that match the text entered. Users with admin rights will be able to enter “tags”, which are custom identifiers, that allow accounts to be grouped based on any desired criteria. Text entered into the search box will find matches in any of the below columns, including tags.

It is important to note that accounts are broken into two main categories, production accounts and test accounts. Production accounts are live trading accounts. Orders entered into test accounts will not interact with the market. To ensure that test users don’t inadvertently select a production account, there is a “Test Mode” toggle which automatically filters for only test accounts.

The “Link account selectors” toggle will link or unlink the account chosen in this menu with the account shown on the order ticket.

User Profile​

Clicking on the user profile icon on the right-side of the Ticker Ribbon will display the current user’s profile information. This section displays the username, their user type, client firm, email address associated with the account, and the URL used. This information can be important when troubleshooting with the SpiderRock Support Group.
