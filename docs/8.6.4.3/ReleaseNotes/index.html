Release Notes

April 18th, 2026 Release Notes​

Message and enum differences can be found here.

SpiderRock Connect:​

  • Added support for non-hex values in groupingCode and riskGroupID on messages sent via FIX through the clientGateway.
  • Added support for one-letter tickers in MLink calculators.
  • Added support for LiveSurfaceComposite record in V8.
  • Adjusted logic to properly leverage numMakeExchange/maxExposureSize from parent limit record.
  • Bid price used as upper bound for srprc when ask price is missing in OptionCloseMark.
  • Dividend overrides are properly applied to option price and vol calculations in MLink.
  • Generate riskMetrics in SpdrRiskExecution for fills bridged from V7.
  • Override to allow routing through NDX options.
  • Relaxed Open Exposure risk check from a hard pre-trade block to a soft gross-violation check, reducing false order rejects.
  • Removed errant Delta on expired OTM options.
  • Stabilization of stock PnL after close.
  • Upgrade to new version of Bloomberg Terminal Connect.

Citrix Tools:​

  • Symbol Viewer - Autohedge settings correctly load on startup.
  • Symbol Viewer - Live surface now appears correctly in Option Calculator before market open.
  • Symbol Viewer - Toggle for Log(Ivol/SPY) in AtmVol Chart now updates correctly when switching symbols from SPY.
  • Symbol Viewer: Ticket Blotter - Added functionality to efficiently set/edit orderActiveSize.
  • Symbol Viewer: Ticket Blotter - Improved synchronization of vol to price conversion between Ticket Blotter and Execution Engine.

WebApps Tool Suite:​

  • Mesa (Portal) - Only able to select a row by clicking the checkbox.
  • Ignite (Trade) - Enhanced AlgoDisplay scaling.
  • Ignite (Trade) - Order cards limited to sorting by create time.
  • Ignite (Trade) - Renamed the orderShape Mix Basket to Basket.
  • Ignite (Trade) - Root selection for options on futures is persistent.
  • Ignite (Trade) - Surface chart is now visible.

March 28th, 2026 Release Notes​

Message and enum differences can be found here.

SpiderRock Connect:​

  • Added OpnDivPnl back into msgRiskCubeDetailv5.
  • Preserves user data from reviewResponse when a Stage Review order gets modified.

Citrix Tools:​

  • Symbol Viewer - Fixed a bug to correct cycle through the risk widgets.
  • Symbol Viewer - Fixed a bug with the OpenCn display.
  • Symbol Viewer - Fixed issue with missing greeks and rates before market open.
  • Symbol Viewer: Ticket Blotter - Increased vega ratio to 10.0.
  • Symbol Viewer: Ticket Blotter (MLeg) - Fixed bug that prevented unknown stock reject when trading SPX/SPXW packages.

WebApps Tool Suite:​

  • Portal - Fixed bug in which filters could not be applied if a record is open.
  • Trade - Added a tab to order queue that displays the previous day's orders.
  • Trade - Added probability columns to options montage.
  • Trade - Added probability parameters for secondary limits.
  • Trade - Can now multiselect and choose order shape directly in Option Montage.
  • Trade - Disabled the "Add Stock" button for European tickers.
  • Trade - Enabled taker MLeg for Europe.
  • Trade - Updated color scheme in position and risk table.
  • Trade - Updated FLEX DAC leg columns.
  • Trade - Updated the order items column to display client configure order parameters.

March 7th, 2026 Release Notes​

Message and enum differences can be found here.

SpiderRock Connect:​

  • Added CBOE ExchAuctionPrint and CBOE ExchAuctionNotice.
  • Added fill theta to omv2 record.
  • Added group hex to SpdrParentExecution.
  • Added HistoricalVolatilitiesHist.
  • Added VegaLeft and $VegaLeft to omv2 aggregations.
  • Enhanced implied pricing model for multi-deliverable options.
  • Fix for some missing CBOE spread quotes.
  • Fixed minor bug in historical surfaceATM archive.
  • Groundwork laid for upcoming V4 Quant Package.
  • Market Limit in ICE chat now displays "PRC".
  • Renamed trdTheta aggregation to match v7's $Theta.
  • SpdrMLegBrkrEvent now properly updates on close events.
  • Split cancel and entry cut-off times for closing orders.

Citrix Tools:​

  • Risk Viewer - Fixed display bugs for weighted vega and div PnL.
  • Risk Manager: Blotter - Added strategy drop down when blotter is opened by the "close delta" or "close position" prompt.
  • Symbol Viewer - Now filters out expired strikes.
  • Symbol Viewer: MLeg Blotter - Added optDGa to option total row.
  • Symbol Viewer: Spread Order Ticket - Max Visible quantity can now be set to a value below 100.
  • Symbol Viewer: Ticket Blotter - Increased vega ratio to match v7.

WebApps​

Trade​

  • Added menu to options montage multiselect.

  • Added scroll feature to basket order items.
  • Allowed disclosed firms' identity to be viewable.

  • Updated uPrc range capability.
