Citrix Tool Suite Troubleshooting Guide​

We've added a new Troubleshooting Instructions section to the Citrix Tool Suite page. This section covers the most common access and display issues encountered when logging into SpiderRock tools, including:

  • URL & login problems — deprecated URLs, bookmark issues
  • MFA troubleshooting — setup, wrong QR codes, password vs. passcode fields
  • Citrix session errors — hung sessions, failed-to-start messages
  • Firewall configuration — required URLs and ports
  • Display issues — DPI settings, Symbol Viewer rendering
